# YourAIGuard

A Firefox extension that helps you stay safe by checking the trustworthiness of AI responses on ChatGPT.

## What it does

- Monitors every ChatGPT response automatically
- Displays a confidence indicator directly below each AI response
- Designed to be simple and easy to use for everyone

## Installation (Development)

1. Open Firefox Developer Edition
2. Go to `about:debugging`
3. Click **"This Firefox"**
4. Click **"Load Temporary Add-on"**
5. Select the `manifest.json` file from this folder

## Supported Platforms

- [x] ChatGPT (chatgpt.com)
- [ ] Claude (coming soon)
- [ ] Gemini (coming soon)
